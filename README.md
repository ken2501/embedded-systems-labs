- Arduino Nano programming
- sensor interfacing
- actuator control
- analogue and digital signal processing

- real-time embedded system behaviour

# Semester 1 Labs

## Lab 1 — LED Sequence Control

Microcontroller-based LED sequencing system using:
- 8 LEDs
- push-button input
- programmable timing behaviour

### Concepts Demonstrated
- digital output control
- user input handling
- sequence logic
- array-based animation systems
- structured embedded programming

### Key Features
- custom LED animation engine
- 2D-array sequence storage
- interactive timing behaviour
- reusable sequence logic

### Main Learning Outcomes
- understanding Arduino program structure
- nested loops and arrays
- embedded timing logic
- hardware/software integration

---

## Lab 2 — Servo Motor Control

Real-time servo motor control system using:
- potentiometer analogue input
- PWM signal generation
- servo actuator positioning
- voltage-dependent LED feedback

### Concepts Demonstrated
- analogue signal acquisition
- pulse-width modulation (PWM)
- actuator control
- real-time proportional mapping
- non-blocking timing using millis()

### Key Features
- live servo angle control
- voltage-reactive LED behaviour
- startup diagnostics sequence
- proportional input scaling

### Main Learning Outcomes
- servo motor interfacing
- PWM fundamentals
- analogue-to-digital conversion
- embedded state behaviour

---

## Lab 3 — Capacitor Charging Behaviour

Experimental analysis of RC charging circuits using an Arduino Nano.

The project measures capacitor voltage over time and compares measured charging curves against theoretical RC models.

### Concepts Demonstrated
- RC circuit behaviour
- exponential charging curves
- analogue voltage measurement
- serial telemetry logging
- theoretical vs measured comparison

### Experimental Variations
Three charging curves were analysed using different:
- resistance values
- capacitance values
- RC time constants

### Key Features
- automated charging measurements
- CSV serial data output
- multi-curve comparison
- real-time voltage sampling
- theoretical model validation

### Main Learning Outcomes
- RC time constant analysis
- experimental measurement methodology
- signal acquisition
- embedded data logging
- practical electronics analysis

---

### System Features

### Concepts Demonstrated
- embedded state machines
- multi-condition logic
- real-time event handling
- safety-state architecture
- interactive embedded systems

### Key Features
- configurable threat levels
- emergency safe-state reset
- visual and audio alerts
- modular system behaviour

### Main Learning Outcomes
- embedded architecture design
- scalable logic systems
- event-driven programming
- embedded system safety concepts
